# Gestionnaire de Tâches - CCP1

Application web responsive de gestion de tâches développée dans le cadre du Titre Professionnel Développeur Web et Web Mobile (CCP1).

## Démonstration en ligne

https://Prasannadl.github.io/gestionnaire-taches-ccp1/

## Description

Application web permettant de gérer ses tâches quotidiennes avec une interface simple et intuitive.

## Fonctionnalités

- Ajouter une nouvelle tâche
- Marquer des tâches comme terminées
- Supprimer des tâches
- Compteur de tâches actives et terminées
- Sauvegarde automatique locale
- Interface responsive (mobile et desktop)

- CSS3
- JavaScript ES6+

## Utilisation

1. Ouvrir l'application dans un navigateur
2. Saisir une tâche dans le champ de texte
3. Cliquer sur "Ajouter"
4. Cocher la case pour marquer une tâche terminée
5. Cliquer sur "Supprimer" pour effacer une tâche
